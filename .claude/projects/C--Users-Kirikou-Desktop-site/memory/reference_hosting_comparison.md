## Free Static Hosting Comparison (researched March 2026)

| Service | URL format | Wipes for inactivity? | Bandwidth | Custom domain free? |
|---|---|---|---|---|
| **GitHub Pages** | `username.github.io` | **Never** | 100 GB/mo | Yes |
| **Cloudflare Pages** | `project.pages.dev` | **Never** | **Unlimited** | Yes |
| **Netlify** | `name.netlify.app` | No known policy | 100 GB/mo | Yes |
| **Vercel** | `name.vercel.app` | **Possible** (discretionary) | 100 GB/mo | Yes |
| **Render** | `name.onrender.com` | No for static sites | 100 GB/mo | Yes (max 2) |

GitHub Pages and Cloudflare Pages are the safest for "set and forget" — neither deletes for inactivity.
Previous hosting on alwaysdata was wiped for inactivity.
User is leaning toward GitHub Pages.
